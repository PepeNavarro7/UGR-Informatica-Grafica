Ejecutar con "make do", "make" solo compila el ejecutable pero no lo llama.
"make clean" para borrar archivos innecesarios

4 modos de dibujado:
1 -> modo vertices
2 -> modo aristas
3 -> modo color sólido 
4 -> modo sólido de colores aleatorios
0 -> modo completo (vertices, aristas y colores aleatorios)
5 -> SOLID_FLAT (normales caras)
6 -> SOLID_SMOOTH (normales_vertices)
7 -> objeto sencillo con textura
		
8 cuerpos geométricos:
C -> Cubo 
E -> Esfera 
L -> Cilindro 
N -> Cono 
O -> Objeto del que se posee el PLY 
P -> Pirámide
R -> Rotacion de un perfil
X -> Extrusion de un polígono

2 modelos jerárquicos articulados:
A -> Jirafa
H -> Excavadora

S -> Activar/Desactivar animacion
I -> Activar/Desactivar segunda luz
Q -> Cerrar

F1-F2 -> Girar cuerpo jirafa
F3-F4 -> Girar cola
F5-F6 -> Girar cuello entero
F7-F8 -> Girar solo parte de arriba del cuello
F9-F10 -> Girar cabeza
F11-F12 -> Girar segunda luz
